üî¢ DAX Formulas Examples

1. Total Premium:

2. Claim Settlement Ratio:

COUNTROWS(Claim)

YoY_Growth =
DIVIDE(
[Total_Premium] - CALCULATE([Total_Premium], SAMEPERIODLASTYEAR(Date[Date])),

[Total_Premium],

üé® Tableau Key Concepts

1. Dimensions vs Measures:

Dimensions (‡§Ü‡§Ø‡§æ‡§Æ): Qualitative, categorical data
Examples: PolicyType, CustomerCity, AgentName, ClaimStatus
Used for: Slicing and filtering data

Measures (‡§Æ‡§æ‡§™): Quantitative, numerical data
Examples: Premium, ClaimAmount, PolicyCount, Age
Used for: Aggregations (SUM, AVG, COUNT)

2. Discrete vs Continuous:

Discrete (‡§µ‡§ø‡§µ‡§ø‡§ï‡•ç‡§§): Individual, separate values (Blue pills in Tableau)
Examples: Years (2021, 2022, 2023), Policy Types

Continuous (‡§∏‡§§‡§§): Range of values (Green pills in Tableau)
Examples: Premium amount, Date range, Temperature

3. Shelves in Tableau:
‚Ä¢ Rows Shelf: Creates vertical axis
‚Ä¢ Columns Shelf: Creates horizontal axis
‚Ä¢ Filters Shelf: Filter data
‚Ä¢ Marks Card: Control color, size, label, detail
‚Ä¢ Pages Shelf: Create animations

üìñ Part 5: Data Storytelling

üé≠ What is Data Storytelling?

Data Storytelling: The art of communicating insights from data using narrative and visualizations to drive action.

3 Key Components:

1. Data (‡§°‡•á‡§ü‡§æ): The facts and figures
Example: Claim settlement ratio dropped from 95% to 88%

2. Narrative (‡§ï‡§•‡§æ): The story and context
Example: "Our claims team is overwhelmed due to 40% increase in claims this quarter..."

3. Visuals (‡§¶‡•É‡§∂‡•ç‡§Ø): Charts and graphics
Example: Line chart showing settlement ratio decline with annotation

üìù Data Storytelling Framework

1. CONTEXT (‡§∏‡§Ç‡§¶‡§∞‡•ç‡§≠): Set the stage
‚Ä¢ What is the business situation?
‚Ä¢ Why should audience care?
‚Ä¢ What decisions need to be made?

2. CHALLENGE (‡§ö‡•Å‡§®‡•å‡§§‡•Ä): Identify the problem
‚Ä¢ What's going wrong?
‚Ä¢ What are the pain points?
‚Ä¢ What's at stake?

3. INSIGHTS (‡§Ö‡§Ç‡§§‡§∞‡•ç‡§¶‡•É‡§∑‡•ç‡§ü‡§ø): Reveal the findings
‚Ä¢ What does the data show?
‚Ä¢ What patterns emerged?
‚Ä¢ What's the root cause?

4. ACTION (‡§ï‡§æ‡§∞‡•ç‡§∞‡§µ‡§æ‡§à): Recommend next steps
‚Ä¢ What should we do?
‚Ä¢ What are the options?
‚Ä¢ What's the expected impact?

üéØ Storytelling Best Practices

DO's ‚úÖ
‚úì Start with the punchline (main insight first)
‚úì Use simple language, avoid jargon
‚úì Focus on 3-5 key points maximum
‚úì Provide context with comparisons (vs last year, vs target)
‚úì Highlight insights with annotations
‚úì End with clear call-to-action
‚úì Use executive summary for busy stakeholders

DON'Ts ‚ùå
‚ùå Don't show every data point you found
‚ùå Don't use complex statistical terms
‚ùå Don't create chart soup (too many visuals)
‚ùå Don't hide the conclusion at the end
‚ùå Don't use 3D charts or unnecessary effects
‚ùå Don't ignore your audience's expertise level
