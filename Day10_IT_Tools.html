What is ETL?

Definition: ETL is the process of extracting data from multiple sources, transforming it into a consistent format, and loading it into a data warehouse.

ETL Process Steps

1. Extract (‡§®‡§ø‡§∑‡•ç‡§ï‡§∞‡•ç‡§∑‡§£): Pull data from source systems
‚Ä¢ Sources: Databases, APIs, CSV files, Excel, CRM systems
‚Ä¢ Methods: Full extraction or Incremental (only new/changed data)
Example: Extract daily policy sales from branch databases
2. Transform (‡§∞‡•Ç‡§™‡§æ‡§Ç‡§§‡§∞‡§£): Clean, standardize, and prepare data
‚Ä¢ Cleaning: Remove duplicates, fix errors, handle nulls
‚Ä¢ Standardization: Convert date formats (DD/MM/YYYY to YYYY-MM-DD)
‚Ä¢ Validation: Check data quality rules
‚Ä¢ Aggregation: Sum, average, count operations
‚Ä¢ Enrichment: Add calculated fields (Age from DOB)
‚Ä¢ Integration: Merge data from multiple sources
3. Load (‡§≤‡•ã‡§°‡§ø‡§Ç‡§ó): Insert transformed data into warehouse
‚Ä¢ Full Load: Replace all data (for small datasets)
‚Ä¢ Incremental Load: Add only new/changed records
‚Ä¢ Real-time Load: Continuous loading as data arrives
