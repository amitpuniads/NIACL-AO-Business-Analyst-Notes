üìä Part 1: Data Interpretation Basics

Types of Data Representation

  1. Tables: Raw data in rows and columns
  2. Bar Charts: Vertical or horizontal bars for comparison
  3. Line Graphs: Shows trends over time
  4. Pie Charts: Shows percentage distribution
  5. Caselets: Data presented in paragraph form

üîë Key Formulas for DI

1. Percentage:
Percentage = (Part / Whole) √ó 100

2. Percentage Change:
% Change = [(New Value - Old Value) / Old Value] √ó 100

3. Profit/Loss:
Profit = Selling Price - Cost Price
Profit % = (Profit / CP) √ó 100

4. Ratio:
If A:B = 3:5, and A = 15, then B = (15 √ó 5) / 3 = 25

5. Average:
Average = Sum of all values / Number of values

üìà Growth Rate & CAGR

Simple Growth Rate:
Growth Rate = [(Value_year2 - Value_year1) / Value_year1] √ó 100

CAGR (Compound Annual Growth Rate):
CAGR = [(Ending Value / Beginning Value)^(1/n) - 1] √ó 100
Where n = number of years
Example 1: Growth Rate
NIACL premium in 2022: ‚Çπ10,000 Cr
NIACL premium in 2023: ‚Çπ12,000 Cr
Growth Rate = [(12000 - 10000) / 10000] √ó 100 = 20%
Example 2: CAGR
Initial premium (2020): ‚Çπ8,000 Cr
Final premium (2024): ‚Çπ15,000 Cr
n = 4 years
CAGR = [(15000/8000)^(1/4) - 1] √ó 100
= [(1.875)^0.25 - 1] √ó 100
= [1.170 - 1] √ó 100 = 17.0%

üìê Important Formulas Quick Reference

Percentage & Profit/Loss:
‚Ä¢ % = (Part/Whole) √ó 100
‚Ä¢ % Change = [(New-Old)/Old] √ó 100
‚Ä¢ Profit% = (Profit/CP) √ó 100
‚Ä¢ SP = CP √ó (100+Profit%)/100
‚Ä¢ CP = SP √ó 100/(100+Profit%)

Interest:
‚Ä¢ SI = (P√óR√óT)/100
‚Ä¢ CI = P[(1+R/100)^T - 1]
‚Ä¢ Difference (CI-SI for 2 years) = P(R/100)¬≤

Time, Speed, Distance:
‚Ä¢ Speed = Distance/Time
‚Ä¢ Average Speed = Total Distance/Total Time
‚Ä¢ Average Speed (same distance) = (2√óS1√óS2)/(S1+S2)
‚Ä¢ Relative Speed (opposite) = S1 + S2
‚Ä¢ Relative Speed (same direction) = |S1 - S2|

Work:
‚Ä¢ Work = Time √ó Efficiency
‚Ä¢ Combined work = 1/A + 1/B
‚Ä¢ M1√óD1√óH1/W1 = M2√óD2√óH2/W2

Ratio & Proportion:
‚Ä¢ If a/b = c/d then ad = bc
‚Ä¢ If a:b = c:d, then (a+b):(a-b) = (c+d):(c-d)
‚Ä¢ Compound ratio of a:b and c:d = ac:bd

Growth Rate:
‚Ä¢ Simple Growth = [(End-Start)/Start] √ó 100
‚Ä¢ CAGR = [(End/Start)^(1/n) - 1] √ó 100
‚Ä¢ Doubling Time ‚âà 72/Rate

Alligation (Mixture):
‚Ä¢ (C-M):(M-D) where C=Costly, D=Cheaper, M=Mean
‚Ä¢ Final quantity = (Q1√óP1 + Q2√óP2)/(Q1+Q2)
